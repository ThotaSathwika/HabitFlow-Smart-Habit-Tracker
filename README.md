# 🌱 HabitFlow – Smart Habit Tracker

HabitFlow is a modern productivity-focused habit tracking application designed to help users build consistency, improve discipline, and monitor self-growth through analytics and gamification.

---

## 🚀 Features

- ✅ Daily Habit Tracking
- 🔥 Streak Monitoring
- 📊 Analytics Dashboard
- 🏆 Achievement System
- 🎯 Goal Tracking
- 📝 Mood Journal
- 📁 CSV Export
- 🔐 Authentication System
- 🌙 Futuristic Glassmorphism UI

---

## 🛠️ Tech Stack

### Backend
- Flask
- SQLite
- Flask-Login
- Flask-Bcrypt
- SQLAlchemy

### Frontend
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
- Chart.js
